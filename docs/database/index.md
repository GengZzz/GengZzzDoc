# 数据库

这里记录数据库相关内容，包括关系型数据库、缓存、文档数据库和搜索引擎。

## 目录

- [MySQL](./mysql/) — 完整的 MySQL 技术文档，涵盖索引、事务、锁、查询优化、分库分表、主从复制等
- [Redis](./redis/) — Redis 数据结构底层实现、持久化、集群、分布式锁、缓存模式等
- [MongoDB](./mongodb/) — MongoDB 文档建模、聚合管道、副本集、分片集群、Change Streams 等
- [PostgreSQL](./postgresql/) — 完整的 PostgreSQL 技术文档，覆盖 MVCC、查询优化器、流复制、分区表、扩展生态等
- [Elasticsearch](./elasticsearch/) — Elasticsearch 技术文档，涵盖倒排索引、分布式搜索、集群管理、性能调优等
