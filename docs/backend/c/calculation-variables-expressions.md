# 计算、变量与表达式

第二阶段的核心是计算。程序需要保存数据、读入数据、处理数据，再输出结果。变量、类型和表达式就是完成这些事情的基本工具。

## 变量

变量是程序中的一块命名存储空间。

```c
int age = 18;
double price = 9.5;
char grade = 'A';
```

变量定义时要说明类型。类型决定它能保存什么样的数据，以及参与运算时如何解释这些数据。

## 输入

```c
#include <stdio.h>

int main(void) {
    int a;
    scanf("%d", &a);
    printf("a = %d\n", a);
    return 0;
}
```

`scanf` 需要变量地址，所以变量名前要写 `&`。这是很多初学者第一次遇到地址概念，先记住：读取输入时，`scanf` 要知道把数据放到哪里。

## 常见类型

| 类型 | 用途 | 输出格式 |
| --- | --- | --- |
| `int` | 整数 | `%d` |
| `double` | 小数 | `%f` |
| `char` | 单个字符 | `%c` |

```c
int count = 3;
double weight = 60.5;
char level = 'B';

printf("%d %.1f %c\n", count, weight, level);
```

## 表达式

表达式会计算出一个值。

```c
int total = 3 + 4 * 5;
double average = (80 + 90 + 75) / 3.0;
```

整数除法要特别小心：

```c
printf("%d\n", 5 / 2);   // 2
printf("%f\n", 5 / 2.0); // 2.500000
```

只要参与运算的一方是小数，结果就会按小数计算。

## 复合赋值与自增

```c
count = count + 1;
count += 1;
count++;
```

三种写法都能让 `count` 增加 1。初学时优先写清楚，确认理解后再使用简写。

## 练习

1. 输入两个整数，输出它们的和、差、积、商。
2. 输入摄氏温度，转换为华氏温度。
3. 输入三门成绩，输出平均分，注意整数除法问题。

## 小结

- 变量保存数据，类型决定解释方式。
- `scanf` 读取数据时通常需要 `&`。
- 表达式用于计算，运算符优先级会影响结果。
- 整数除法和浮点除法结果不同。
