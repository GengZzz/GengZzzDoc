# 指针与字符串

指针和字符串是 C 语言入门后期最重要的主题。它们让你更接近内存，也更容易犯边界错误。

## 指针是什么

指针变量保存地址。

```c
int a = 10;
int* p = &a;

printf("%d\n", *p);
```

- `&a` 取得变量 `a` 的地址。
- `p` 保存这个地址。
- `*p` 访问这个地址上的值。

## 指针和函数

如果希望函数修改调用者传入的变量，可以传指针。

```c
void swap(int* a, int* b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int x = 3;
int y = 5;
swap(&x, &y);
```

这里 `swap` 拿到的是 `x` 和 `y` 的地址，所以能修改它们。

## 字符类型

```c
char ch = 'A';
printf("%c %d\n", ch, ch);
```

字符本质上也可以看作整数编码。理解这一点有助于处理字符判断和转换。

## 字符串

C 字符串是以 `'\0'` 结尾的字符数组。

```c
char name[] = "Alice";
```

实际存储为：

```text
A l i c e \0
```

字符串函数通常依赖这个结束标记。如果没有 `'\0'`，函数就不知道该在哪里停止。

## 常用字符串函数

```c
#include <string.h>

strlen(name);        // 长度
strcmp(a, b);        // 比较
strcpy(dest, source); // 复制
strcat(dest, source); // 拼接
```

使用 `strcpy`、`strcat` 时要确认目标数组空间足够，否则可能越界。

## 练习

1. 写一个 `swap` 函数交换两个整数。
2. 输入一个字符串，统计其中字母个数。
3. 不使用 `strlen`，自己计算字符串长度。

## 小结

- 指针保存地址，`*` 可以访问地址上的值。
- 传指针可以让函数修改外部变量。
- C 字符串以 `'\0'` 结束。
- 字符串处理最重要的是空间大小和结束标记。
