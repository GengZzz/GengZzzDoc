# 循环控制与枚举

循环控制关注的不是“会不会写循环”，而是“如何让循环在正确的时候继续、跳过或停止”。翁恺课程中很多入门题，本质都是枚举与循环控制。

## break

`break` 用来提前结束循环。

```c
for (int i = 1; i <= 100; i++) {
    if (i % 17 == 0) {
        printf("%d\n", i);
        break;
    }
}
```

这个程序找到第一个能被 17 整除的数后就停止。

## continue

`continue` 用来跳过本轮循环后面的语句，进入下一轮。

```c
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;
    }
    printf("%d\n", i);
}
```

输出所有奇数。

## 多重循环

```c
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        printf("%d*%d=%d ", j, i, i * j);
    }
    printf("\n");
}
```

多重循环要分清每一层循环负责什么。外层通常控制行，内层通常控制列。

## 枚举

枚举就是把可能的答案一个个试出来。

```c
for (int x = 0; x <= 20; x++) {
    for (int y = 0; y <= 33; y++) {
        int z = 100 - x - y;
        if (5 * x + 3 * y + z / 3.0 == 100) {
            printf("%d %d %d\n", x, y, z);
        }
    }
}
```

这类题重点不是套公式，而是确定搜索范围、写出约束条件、避免遗漏。

## 循环调试方法

当循环结果不对时，可以临时打印循环变量：

```c
printf("i=%d sum=%d\n", i, sum);
```

看见每一轮状态变化，比盯着代码猜更可靠。

## 练习

1. 输出九九乘法表。
2. 找出 1 到 1000 中第一个能同时被 7 和 13 整除的数。
3. 枚举两位数，找出十位和个位之和等于 10 的数。

## 小结

- `break` 结束循环，`continue` 跳过本轮。
- 多重循环要明确每一层的职责。
- 枚举题先确定范围，再写约束。
- 循环调试最有效的方法是打印中间状态。
