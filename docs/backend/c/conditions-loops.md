# 判断与循环

判断和循环让程序从“顺序执行”进入“根据情况执行”和“重复执行”。这也是程序开始表达算法的地方。

## if 判断

```c
if (score >= 60) {
    printf("pass\n");
} else {
    printf("fail\n");
}
```

判断条件的结果只有两类：真或假。C 语言中，0 表示假，非 0 表示真。

## 关系运算和逻辑运算

| 运算符 | 含义 |
| --- | --- |
| `==` | 相等 |
| `!=` | 不相等 |
| `>`、`>=` | 大于、大于等于 |
| `<`、`<=` | 小于、小于等于 |
| `&&` | 并且 |
| `||` | 或者 |
| `!` | 非 |

```c
if (age >= 18 && score >= 60) {
    printf("allowed\n");
}
```

注意，判断相等使用 `==`，赋值使用 `=`。

## 多路分支

```c
if (score >= 90) {
    printf("A\n");
} else if (score >= 80) {
    printf("B\n");
} else if (score >= 60) {
    printf("C\n");
} else {
    printf("D\n");
}
```

多路分支要注意顺序。上面的条件从高到低判断，如果先写 `score >= 60`，后面的高分段就永远进不去了。

## while 循环

```c
int n = 1;
while (n <= 5) {
    printf("%d\n", n);
    n++;
}
```

写循环时先问三个问题：

1. 循环变量初始值是什么？
2. 什么时候继续循环？
3. 每轮循环后状态如何变化？

## for 循环

```c
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
```

`for` 适合循环次数明确的场景。`while` 适合“不知道循环多少次，但知道什么时候停”的场景。

## 练习

1. 输入一个数，判断是正数、负数还是 0。
2. 输出 1 到 100 中所有偶数。
3. 输入若干整数，以 0 结束，输出总和。

## 小结

- 判断让程序根据条件选择路径。
- 循环让程序重复执行一段逻辑。
- 写循环时一定要确认状态会变化，否则可能死循环。
- 多路分支要注意判断顺序。
