# C++

这套 C++ 大纲面向从入门到能写工程代码的读者。它参考了菜鸟教程的知识覆盖面，也借鉴《C++ Primer Plus》的学习推进方式：先让读者能写、能运行、能读懂程序，再逐步进入数据组织、函数抽象、对象模型、模板、STL、异常、文件、现代 C++ 和工程实践。

## 编排原则

大纲不按“第几章、第几课”编号，而按读者理解递进来组织。每一页都尽量回答三个问题：这个主题解决什么问题、读者应该掌握哪些概念、写代码时最容易踩什么坑。

## 学习路径

### 入门阶段

先完成 [起步与开发环境](./getting-started.md)、[语法、数据与表达式](./syntax-data-expressions.md)、[复合类型与内存基础](./compound-types-memory.md) 和 [控制流与程序逻辑](./control-flow.md)。这个阶段的目标不是背语法，而是能把一个小问题拆成输入、计算和输出。

### 能力成型阶段

继续学习 [函数与模块化](./functions-modular.md)、[指针、引用与资源管理](./pointers-references-resources.md)、[类与对象](./classes-objects.md)、[继承与多态](./inheritance-polymorphism.md)。这个阶段开始从“写语句”转向“组织程序”。

### 进阶实践阶段

最后进入 [模板与泛型编程](./templates-generic.md)、[STL 与模板](./stl-templates.md)、[输入输出、文件与异常](./io-files-exceptions.md)、[现代 C++](./modern-cpp.md)、[并发编程](./concurrency.md)、[数据结构与算法实践](./algorithms-data-structures.md) 和 [工程实践](./engineering-practice.md)。
